# Carrito de Compras en Java 🛒

Aplicación web desarrollada con Java (Servlets, JSP) y MySQL que permite gestionar productos y un carrito de compras.

## 🚀 Funcionalidades
- Listado de productos
- Gestión de carrito de compras
- Conexión a base de datos
- Arquitectura MVC

## 🛠️ Tecnologías
- Java
- Servlets
- JSP
- MySQL
- JDBC

## 📌 Estructura
- Controlador (Servlets)
- Modelo (Clases Java)
- DAO (Acceso a datos)
- Configuración de conexión

## ▶️ Cómo ejecutar
1. Configurar base de datos MySQL
2. Importar el proyecto en NetBeans
3. Ejecutar en servidor (Tomcat)
